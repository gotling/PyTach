PyTach
================
Python library for communicating with GlobalCache iTach IR/RF device

Usage
================
### Control device
	pytach.py nexa (1-4) (on | off)
	pytach.py multibrackets (up | stop | down)
	pytach.py yamaha (cd | dtv | stereo | vol_up | vol_down)

### Start web server
	pytach.py --web

Features
================
* Command line interface
* REST interface
* Web interface
